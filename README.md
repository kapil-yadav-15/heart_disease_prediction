# heart_disease_prediction-

Built a complete end-to-end Machine Learning pipeline to predict heart disease in patients using a structured medical dataset. The project covers thorough Exploratory Data Analysis (EDA), 
advanced data cleaning, feature engineering, and training and comparing five classification models to identify the best performing algorithm for binary health outcome prediction.

Models Trained & Compared
1. Logistic Regression — Linear baseline classifier predicting survival probability using sigmoid function
2. K-Nearest Neighbors (KNN) — Distance-based classifier; best model selected and saved using joblib for deployment
3. Gaussian Naive Bayes — Probabilistic classifier based on Bayes theorem assuming Gaussian feature distribution
4. Decision Tree Classifier — Non-linear tree-based model splitting on feature thresholds for classification
5. Support Vector Machine — SVM (RBF Kernel) — Margin-based classifier using RBF kernel with probability estimation enabled for robust boundary classification
All five models evaluated on Accuracy Score and F1 Score for comprehensive performance comparison.
